2. Create `eslint.config.mjs` at repo root using the ESLint v9 flat config API with typescript-eslint v8 `projectService: true` for typed linting:

3. Add `lint-staged` configuration to root `package.json`:

   Do NOT add test execution to lint-staged — tests are slow and belong in CI, not pre-commit.
