## OBS-04: SES DNS Verification

### Required DNS Records
1. **SES Domain Identity**: Verify domain in SES console → generates DKIM CNAME records
2. **SPF**: TXT record on domain: `v=spf1 include:amazonses.com ~all`
3. **DKIM**: 3 CNAME records auto-generated by SES (Easy DKIM)
4. **DMARC**: TXT record `_dmarc.thirdwavebbq.com.au`: `v=DMARC1; p=quarantine; rua=mailto:dmarc@thirdwavebbq.com.au`

### Verification Approach
- Use AWS SDK `GetEmailIdentityCommand` to check domain verification status programmatically
- Use Node.js `dns.promises` to check SPF, DKIM, DMARC records from code
- Create a standalone script `scripts/verify-ses-dns.ts` that can be run pre-deploy
